# Network Infrastructure Labs

Hands-on Cisco Packet Tracer simulations focused on practical routing, switching, security, and enterprise network deployments.

This repository contains structured lab scenarios designed to simulate real-world infrastructure implementations. Each lab includes full configurations, verification outputs, troubleshooting notes, and the original Packet Tracer file.

---

## Technologies Covered

- VLAN Deployments
- Inter-VLAN Routing
- Static Routing
- OSPF (Single & Multi-Area)
- EIGRP
- NAT & DHCP
- Access Control Lists (ACL)
- SSH Hardening
- Port Security
- Redundant Topologies
- Enterprise Multi-Site Designs

---

## Repository Structure

Each lab follows a consistent documentation format:

/category/lab-name/
README.md
topology.png
addressing-table.md
device-configurations.txt
verification.txt
troubleshooting-notes.md
lab-file.pkt


---

## Objective

To build production-style network infrastructure simulations and improve practical routing, switching, and troubleshooting skills through structured documentation and configuration.
